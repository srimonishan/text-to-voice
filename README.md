# 🎙️ VoiceForge AI

Transform text into studio-quality voice using ElevenLabs API.

## 🚀 Overview

VoiceForge AI is a production-ready web application that converts text into high-quality, natural-sounding voice in real time. This project demonstrates how AI voice technology can be used for content creation, education, and storytelling.

This project was built as part of my exploration and application of ElevenLabs API in real-world use cases.

## ⚡ Features

- Text-to-Speech generation using ElevenLabs API
- Multiple voice selection
- Adjustable voice settings (stability, similarity)
- Content modes (Motivational, Storytelling, Educational)
- Audio playback + download
- Clean and responsive UI

## 🧠 Why This Project

As an AI Product Manager at EdgeOS Health Institute, I have worked with ElevenLabs in production environments for content generation and AI-driven workflows.

This project is a public demonstration of how ElevenLabs API can be used to build real-world AI applications for creators, developers, and educators.

## 🛠️ Tech Stack

- Node.js (Express)
- HTML, CSS, JavaScript
- ElevenLabs Text-to-Speech API
